webapp
======

在WEB上构建类似NativeAPP体验的应用

利用input类型checkbox来实现一些简单的交互，减少JS事件
例如：
    <!--<style>
        .input0:checked ~ .div0 {
            color:red;
        }
    </style>
    <input id="input0" type="checkbox">
    <div class="div0">这是一个测试</div>-->
    点击该多选框，‘这是一个测试’字体颜色变为红色。
    以此类推，结合最新的CSS3属性，可以实现更多的交互。
